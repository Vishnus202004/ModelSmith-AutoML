<strong>ModelSmith AutoML</strong> is an advanced, fully autonomous Human-in-the-Loop Machine Learning platform that bridges the gap between Data Science and Software Engineering.

<h2>Workflow</h2>

<ol>

<li>
<strong>Upload & Analyze</strong><br>
Upload a raw CSV dataset. The backend parses the file and returns a structural profile to the Data Health Studio.
</li>

<br>

<li>
<strong>Pre-Process</strong><br>
Configure target columns, remove irrelevant features, and define missing-value imputation strategies.
</li>

<br>

<li>
<strong>Train & Optimize</strong><br>
The engine preprocesses the dataset, trains six machine learning models, selects the best performer, and launches an Optuna optimization cycle.
</li>

<br>

<li>
<strong>Evaluate & Explain</strong><br>
View Accuracy, R² Score, Feature Importance, Histograms, and AI-generated explanations powered by Groq Llama 3.1.
</li>

<br>

<li>
<strong>Deploy</strong><br>
Generate Dockerized FastAPI microservices ready for production deployment.
</li>

</ol>
